Average loan rate by occupation and income level

Do pilots face sky-high levels of interest? Are bus drivers taken for a ride?

The following analysis allows you to find out. I took data from Prosper , which is a peer-to-peer lending site. This data was sourced from Udacity. It had data on 113,937 loans with 81 variables on each loan, including loan amount, borrower rate (or interest rate), current loan status, borrower income, and many others. The data was last updated 03/11/2014. These loans are all unsecured loans.

I decided to focus on occupation and income level. Please use the graph below to explore whether different occupations face different loan rates (APRs) even when they are in the same income bracket.

How to interpret and use the graph

  • I have calculated the average APR for each occupation and each income group
  • Each occupation has a different APR profile
  • To examine each, either run the animation, or select an occupation from the list on the right
  • To see precise details for each occupation and income level, use the tool-tip
  • The overall average for the entire group is shown in black
  • For more details on the data preparation, see below the chart

Data source

This data was sourced from Udacity, but ultimately came from Prosper

Data preparation

I have focused on the APR and income groupings. I wrangled the data in R first, and amalgamated some occupations, reducing the total number of different occupations from 66 to 50.
